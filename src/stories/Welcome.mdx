# ISP-ADMIN-UI-KIT

Библиотека для **React**, предоставляющая готовый функциональный каркас для административных панелей.
Позволяет ускорить разработку благодаря стандартным компонентам и страницам с поддержкой темизации и навигации.

## Основные возможности

| Возможность                     | Описание |
|---------------------------------|----------|
| **Лейаут**                       | Готовая структура для админ-панелей с поддержкой навигации и тем. |
| **Базовый набор страниц**        | Стандартные страницы: "Вход", "Профиль", "404". Можно использовать сразу или адаптировать под проект. |

## Набор предоставляемых страниц

### Приложения

Страница для управления приложениями и их конфигурацией.

### Доступы приложений

Страница для настройки методов доступа приложений.

### Модули

Интерфейс для настройки конфигураций модулей, доступных в системе.

### Пользователи и роли

Группа страниц для управления пользователями и их ролями.

| Страница                | Описание                                              |
| ----------------------- | ----------------------------------------------------- |
| Пользователи            | Список всех пользователей системы                     |
| Пользовательские сессии | Просмотр активных и завершенных сессий пользователей  |
| Просмотр журналов ИБ    | Просмотр событий информационной безопасности          |
| Роли                    | Управление ролями: создание, редактирование, удаление |
