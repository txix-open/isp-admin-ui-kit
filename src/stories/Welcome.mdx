# ISP-ADMIN-UI-KIT
Библиотека для React, предоставляющая готовый функциональный каркас для создания административных панелей. Она ускоряет разработку, предоставляя стандартные компоненты и страницы с поддержкой темизации и навигации.

## Основные возможности

- **Лейаут**: Готовая структура для административных панелей, включающая поддержку навигации и тем.
- **Базовый набор страниц**: Включает стандартные страницы, такие как "Вход", "Профиль", "404", которые можно использовать сразу или адаптировать под свои нужды.

## Подключение библиотеки
1. установить библиотеку в проект
```
npm install isp-admin-ui-kit
```
2. Выполнить шаги из ```README.md```

## Набор предоставляемых страниц

### Приложения
Раздел для управления настройками приложений и их конфигурацией.

### Доступы приложений
Раздел для настройки методов, доступных приложению.

### Модули
Интерфейс для настройки конфигураций модулей, доступных в системе.

### Пользователи и роли
Группа страниц для управления пользователями и их ролями в системе.

#### Пользователи
Страница со списком всех пользователей системы.

#### Пользовательские сессии
Журнал для просмотра активных и завершенных пользовательских сессий.

#### Просмотр журналов ИБ
Интерфейс для просмотра событий информационной безопасности, связанных с действиями пользователей.

#### Роли
Страница для управления ролями пользователей, включая создание, редактирование и удаление.
